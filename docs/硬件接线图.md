## 🔌 详细接线说明

### 1️⃣ OV2640 CSI 摄像头

**连接方式：** 通过 CSI 接口直接连接开发板

```
OV2640 摄像头模块
┌─────────────┐
│  ┌────────┐ │
│  │ OV2640 │ │
│  │ 传感器 │ │
│  └────────┘ │
│             │
│  排线接口   │
│  (24 pin)   │
└──────┬──────┘
       │
       ↓
[开发板 CSI 接口]

系统识别为：/dev/video0
```

**注意事项：**
- ✅ 使用 OV2640 CSI 摄像头（非 USB 摄像头）
- ✅ 排线方向要正确（金属触点朝向板子）
- ✅ 分辨率设置为 640x480 以优化性能
- ❌ 不要使用 USB 摄像头（会占用 CPU 资源）

**设备树配置：**
- 摄像头使用 GPIO1_IO02 和 GPIO1_IO04 作为电源/复位控制
- 这两个引脚已被摄像头占用，不能用于其他模块

### 2️⃣ RC522 RFID 模块

**引脚定义：**

```
RC522 模块（俯视图）
┌─────────────────┐
│  ○ ○ ○ ○ ○ ○ ○ │  ← 8 个引脚
│  V R S S S S I  │
│  C S I C M I R  │
│  C T K L I O Q  │
│     _   _       │
│     C   M       │
│     S   O       │
└─────────────────┘
```

**接线表：**

| RC522 引脚 | 连接到 I.MX6ULL | JP6 引脚 | 线色建议 | 说明 |
|-----------|----------------|---------|---------|------|
| **VCC** | 3.3V | - | 🔴 红色 | ⚠️ **必须接 3.3V，不要接 5V！** |
| **GND** | GND | - | ⚫ 黑色 | 地线 |
| **RST** | 3.3V（直连） | - | 🟡 黄色 | 复位引脚（始终拉高，设备树未配置 RST GPIO） |
| **SPI_CS** | **GPIO1_IO26** | **Pin 6** | 🟠 橙色 | SPI 片选（UART3_CTS_B 复用） |
| **SPI_SCK** | ECSPI3_SCLK | **Pin 11** | 🔵 蓝色 | SPI 时钟 |
| **SPI_MISO** | ECSPI3_MISO | **Pin 10** | 🟢 绿色 | SPI 主入从出 |
| **SPI_MOSI** | ECSPI3_MOSI | **Pin 9** | 🟣 紫色 | SPI 主出从入 |
| **IRQ** | 不接 | - | - | 中断引脚（本项目未使用） |

**接线示意图：**

```
RC522 模块                    I.MX6ULL 开发板
┌─────────┐                  ┌─────────────────┐
│         │                  │                 │
│    VCC ─┼────── 🔴 ──────→ 3.3V             │
│         │                  │                 │
│    GND ─┼────── ⚫ ──────→ GND              │
│         │                  │                 │
│    RST ─┼────── 🟡 ──────→ 3.3V（直连拉高） │
│         │                  │                 │
│ SPI_CS ─┼────── 🟠 ──────→ GPIO1_IO26       │
│         │                  │  (JP6 Pin 6)    │
│         │                  │                 │
│SPI_SCK ─┼────── 🔵 ──────→ ECSPI3_SCLK      │
│         │                  │  (JP6 Pin 11)   │
│         │                  │                 │
│SPI_MISO┼────── 🟢 ──────→ ECSPI3_MISO      │
│         │                  │  (JP6 Pin 10)   │
│         │                  │                 │
│SPI_MOSI┼────── 🟣 ──────→ ECSPI3_MOSI      │
│         │                  │  (JP6 Pin 9)    │
│         │                  │                 │
│    IRQ ─┼────── ✗  不接    │                 │
│         │                  │                 │
└─────────┘                  └─────────────────┘
```

**⚠️ 重要警告：**
- **RC522 工作电压是 3.3V**，接 5V 会烧毁模块！
- SPI 引脚不要接反（MISO 和 MOSI 容易混淆）
- CS 引脚必须接 **GPIO1_IO26**（JP6 **Pin 6**，即 UART3_CTS_B 引脚，设备树中已配置）
- RST 引脚直接接 3.3V 拉高即可（设备树未配置 RST GPIO）
- **注意：** 旧版本使用 GPIO1_IO10 (Pin 14)，但因为与摄像头引脚冲突，已更改为 GPIO1_IO26 (Pin 6)

### 3️⃣ SG90 舵机

**引脚定义：**

```
SG90 舵机
┌─────────────┐
│  ┌────────┐ │
│  │  舵机  │ │
│  │  本体  │ │
│  └────────┘ │
│             │
│  线引出：    │
│  🟫 棕色    │ → GND
│  🔴 红色    │ → 5V (VCC)
│  🟠 橙色    │ → PWM 信号
└─────────────┘
```

**接线表：**

| 舵机线色 | 连接到 I.MX6ULL | JP6 引脚 | 说明 |
|---------|----------------|---------|------|
| **棕色** | GND | - | 地线 |
| **红色** | 5V | - | 电源（需要 5V，电流约 200mA） |
| **橙色** | **GPIO1_IO27** | **Pin 5** | 软件 PWM 控制信号（UART3_RTS_B 复用） |

**接线示意图：**

```
SG90 舵机                     I.MX6ULL 开发板
┌─────────┐                  ┌─────────────────┐
│         │                  │                 │
│   棕色 ─┼────── ⚫ ──────→ GND              │
│         │                  │                 │
│   红色 ─┼────── 🔴 ──────→ 5V               │
│         │                  │                 │
│   橙色 ─┼────── 🟠 ──────→ GPIO1_IO27       │
│         │                  │  (JP6 Pin 5)    │
│         │                  │                 │
└─────────┘                  └─────────────────┘
```

**⚠️ 注意事项：**
- **舵机需要 5V 电源**，不要接 3.3V（会转动无力）
- 5V 引脚要能提供足够电流（至少 200mA）
- 如果开发板 5V 输出不够，可以使用外部 5V 电源（GND 要共地）
- PWM 信号线接 **GPIO1_IO27**（JP6 **Pin 5**，即 UART3_RTS_B 引脚）
- **注意：** 旧版本使用 GPIO5_IO10 (Pin 28)，但因为与摄像头引脚冲突，已更改为 GPIO1_IO27 (Pin 5)

**PWM 控制说明：**
- 使用软件 PWM（内核线程生成）
- PWM 周期：20ms（50Hz）
- 脉宽范围：0.5ms ~ 2.5ms
  - 0.5ms → 0度（关闭位置）
  - 1.5ms → 90度（中间位置）
  - 2.5ms → 180度（全开位置）

### 4️⃣ IR 红外传感器

**引脚定义：**

```
IR 红外传感器模块
┌─────────────┐
│  ┌───────┐  │
│  │ 红外  │  │
│  │ 发射  │  │
│  │ 接收  │  │
│  └───────┘  │
│             │
│  VCC OUT GND│  ← 3 个引脚
│   │   │   │ │
└───┼───┼───┼─┘
    │   │   │
    ↓   ↓   ↓
  3.3V 信号 地
```

**接线表：**

| 传感器引脚 | 连接到 I.MX6ULL | JP6 引脚 | 线色建议 | 说明 |
|-----------|----------------|---------|---------|------|
| **VCC** | 3.3V | - | 🔴 红色 | 电源（3.3V 或 5V 都可以） |
| **OUT** | **GPIO5_IO02** | **Pin 22** | 🟡 黄色 | 信号输出（0=无物体，1=检测到物体） |
| **GND** | GND | - | ⚫ 黑色 | 地线 |

**接线示意图：**

```
IR 传感器                     I.MX6ULL 开发板
┌─────────┐                  ┌─────────────────┐
│         │                  │                 │
│    VCC ─┼────── 🔴 ──────→ 3.3V             │
│         │                  │                 │
│    OUT ─┼────── 🟡 ──────→ GPIO5_IO02       │
│         │                  │  (JP6 Pin 22)   │
│         │                  │                 │
│    GND ─┼────── ⚫ ──────→ GND              │
│         │                  │                 │
└─────────┘                  └─────────────────┘
```

**使用说明：**
- 传感器前方有物体时，OUT 输出 **1**（高电平）
- 传感器前方无物体时，OUT 输出 **0**（低电平）
- 可以通过模块上的电位器调节检测距离
- 用于检测门前是否有人，自动启动人脸识别

### 5️⃣ 板载蜂鸣器

**说明：** I.MX6ULL 开发板自带蜂鸣器，无需额外接线

**控制方式：**
```bash
# 通过 sysfs 控制
echo 1 > /sys/class/leds/beep/brightness  # 开启
echo 0 > /sys/class/leds/beep/brightness  # 关闭
```

**控制引脚：** GPIO5_IO01（JP6 Pin 30）

---

### 6️⃣ 板载麦克风（WM8960）

**说明：** I.MX6ULL 开发板板载 WM8960 音频芯片和麦克风，无需额外接线

**连接方式：**
- 麦克风通过 LINPUT1 和 LINPUT2 差分输入连接到 WM8960
- WM8960 通过 I2C 和 I2S 接口连接到 I.MX6ULL

**使用方式：**
```bash
# 测试录音
arecord -f S16_LE -r 16000 -d 3 test.wav

# 测试播放
aplay test.wav
```

**注意事项：**
- 麦克风增益已在程序初始化时自动配置为最大值
- 使用差分输入（LINPUT1 负极，LINPUT2 正极）提高抗干扰能力

## ✅ 接线检查清单

在通电前，请逐项检查：

### RC522 RFID 模块
- [ ] VCC 接 3.3V（不是 5V！）
- [ ] GND 接 GND
- [ ] RST 接 3.3V（直连拉高）
- [ ] SPI_CS 接 **GPIO1_IO26 (JP6 Pin 6)**
- [ ] SPI_SCK 接 ECSPI3_SCLK (JP6 Pin 11)
- [ ] SPI_MISO 接 ECSPI3_MISO (JP6 Pin 10)
- [ ] SPI_MOSI 接 ECSPI3_MOSI (JP6 Pin 9)
- [ ] IRQ 不接

### SG90 舵机
- [ ] 棕色线接 GND
- [ ] 红色线接 5V（不是 3.3V！）
- [ ] 橙色线接 **GPIO1_IO27 (JP6 Pin 5)**

### IR 传感器
- [ ] VCC 接 3.3V
- [ ] OUT 接 GPIO5_IO02 (JP6 Pin 22)
- [ ] GND 接 GND

### 物理按键（板载 KEY0）
- [ ] 无需额外接线，板载按钮直接使用
- [ ] GPIO1_IO18 (JP6 Pin 18)，按下为低电平

### OV2640 摄像头
- [ ] 排线正确插入 CSI 接口
- [ ] 排线方向正确（金属触点朝向板子）

### 通用检查
- [ ] 所有 GND 已正确连接
- [ ] 电源引脚没有接反
- [ ] 接线牢固，没有松动
- [ ] 没有短路（用万用表检查）

## 🎯 通电测试步骤

### 第一步：只接电源和串口

```bash
# 先不接任何模块，只连接：
```

## ⚠️ 安全警告

### 电源安全
- ❌ **不要将 3.3V 模块接到 5V 电源**（会烧毁）
- ❌ **不要将 5V 模块接到 3.3V 电源**（工作不正常）
- ✅ 确认每个模块的工作电压后再连接

### 接线安全
- ❌ **不要在通电状态下插拔接线**（可能损坏开发板）
- ✅ 接线前先断电，接好后再通电
- ✅ 使用杜邦线时确保接触良好

### 静电防护
- ⚡ 接触开发板前先触摸金属物体放电
- ⚡ 避免在干燥环境下操作
- ⚡ 可以佩戴防静电手环

## 📊 引脚变更历史

### v2.0 (2026-05-24) - 当前版本

**主要变更：**
1. **摄像头改为 OV2640 CSI 摄像头**（非 USB 摄像头）
2. **RC522 CS 引脚更改**：GPIO1_IO10 (Pin 14) → **GPIO1_IO26 (Pin 6)**
   - 原因：GPIO1_IO10 (JTAG_MOD) 与摄像头电源控制引脚冲突
3. **舵机 PWM 引脚更改**：GPIO5_IO10 (Pin 28) → **GPIO1_IO27 (Pin 5)**
   - 原因：GPIO5_IO10 (BOOT_MODE0) 与摄像头复位引脚冲突
4. **禁用 UART3 和 CAN1**：释放 Pin 5 和 Pin 6 给舵机和 RC522 使用
5. **添加板载麦克风说明**：WM8960 音频芯片和麦克风

### v1.0 (2026-01-22) - 初始版本

- 使用 USB 摄像头
- RC522 CS 使用 GPIO1_IO10 (Pin 14)
- 舵机 PWM 使用 GPIO5_IO10 (Pin 28)
