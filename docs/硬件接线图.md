## 🔌 详细接线说明

### 1️⃣ USB 摄像头

**连接方式：** 直接插入 USB 接口

```
USB 摄像头
┌─────────┐
│  USB    │──────────→ [开发板 USB 接口]
│  插头   │
└─────────┘

系统自动识别为：/dev/video0
```

**注意事项：**
- ✅ 使用 UVC 兼容的 USB 摄像头
- ✅ 建议使用 720p 或 1080p 摄像头
- ❌ 不要使用需要特殊驱动的摄像头

---

### 2️⃣ RC522 RFID 模块

**引脚定义：**

```
RC522 模块（俯视图）
┌─────────────────┐
│  ○ ○ ○ ○ ○ ○ ○ │  ← 8 个引脚
│  V R S S S S I  │
│  C S I C M I R  │
│  C T K L I O Q  │
│     _   _       │
│     C   M       │
│     S   O       │
└─────────────────┘
```

**接线表：**

| RC522 引脚 | 连接到 I.MX6ULL | 线色建议 | 说明 |
|-----------|----------------|---------|------|
| **VCC** | 3.3V | 🔴 红色 | ⚠️ **必须接 3.3V，不要接 5V！** |
| **GND** | GND | ⚫ 黑色 | 地线 |
| **RST** | GPIO1_IO09 | 🟡 黄色 | 复位引脚 |
| **SPI_CS** | GPIO1_IO08 | 🟠 橙色 | SPI 片选 |
| **SPI_SCK** | ECSPI3_SCLK | 🔵 蓝色 | SPI 时钟 |
| **SPI_MISO** | ECSPI3_MISO | 🟢 绿色 | SPI 主入从出 |
| **SPI_MOSI** | ECSPI3_MOSI | 🟣 紫色 | SPI 主出从入 |
| **IRQ** | 不接 | - | 中断引脚（本项目未使用） |

**接线示意图：**

```
RC522 模块                    I.MX6ULL 开发板
┌─────────┐                  ┌─────────────────┐
│         │                  │                 │
│    VCC ─┼────── 🔴 ──────→ 3.3V             │
│         │                  │                 │
│    GND ─┼────── ⚫ ──────→ GND              │
│         │                  │                 │
│    RST ─┼────── 🟡 ──────→ GPIO1_IO09       │
│         │                  │                 │
│ SPI_CS ─┼────── 🟠 ──────→ GPIO1_IO08       │
│         │                  │                 │
│SPI_SCK ─┼────── 🔵 ──────→ ECSPI3_SCLK      │
│         │                  │                 │
│SPI_MISO┼────── 🟢 ──────→ ECSPI3_MISO      │
│         │                  │                 │
│SPI_MOSI┼────── 🟣 ──────→ ECSPI3_MOSI      │
│         │                  │                 │
│    IRQ ─┼────── ✗  不接    │                 │
│         │                  │                 │
└─────────┘                  └─────────────────┘
```

**⚠️ 重要警告：**
- **RC522 工作电压是 3.3V**，接 5V 会烧毁模块！
- SPI 引脚不要接反（MISO 和 MOSI 容易混淆）
- CS 引脚必须接 GPIO1_IO08（设备树中已配置）

### 3️⃣ SG90 舵机

**引脚定义：**

```
SG90 舵机
┌─────────────┐
│  ┌────────┐ │
│  │  舵机  │ │
│  │  本体  │ │
│  └────────┘ │
│             │
│  线引出：    │
│  🟫 棕色    │ → GND
│  🔴 红色    │ → 5V (VCC)
│  🟠 橙色    │ → PWM 信号
└─────────────┘
```

**接线表：**

| 舵机线色 | 连接到 I.MX6ULL | 说明 |
|---------|----------------|------|
| **棕色** | GND | 地线 |
| **红色** | 5V | 电源（需要 5V，电流约 200mA） |
| **橙色** | GPIO1_IO06 | PWM 控制信号 |

**接线示意图：**

```
SG90 舵机                     I.MX6ULL 开发板
┌─────────┐                  ┌─────────────────┐
│         │                  │                 │
│   棕色 ─┼────── ⚫ ──────→ GND              │
│         │                  │                 │
│   红色 ─┼────── 🔴 ──────→ 5V               │
│         │                  │                 │
│   橙色 ─┼────── 🟠 ──────→ GPIO1_IO06       │
│         │                  │                 │
└─────────┘                  └─────────────────┘
```

**⚠️ 注意事项：**
- **舵机需要 5V 电源**，不要接 3.3V（会转动无力）
- 5V 引脚要能提供足够电流（至少 200mA）
- 如果开发板 5V 输出不够，可以使用外部 5V 电源（GND 要共地）
- PWM 信号线接 GPIO1_IO06（设备树中已配置）

### 4️⃣ IR 红外传感器

**引脚定义：**

```
IR 红外传感器模块
┌─────────────┐
│  ┌───────┐  │
│  │ 红外  │  │
│  │ 发射  │  │
│  │ 接收  │  │
│  └───────┘  │
│             │
│  VCC OUT GND│  ← 3 个引脚
│   │   │   │ │
└───┼───┼───┼─┘
    │   │   │
    ↓   ↓   ↓
  3.3V 信号 地
```

**接线表：**

| 传感器引脚 | 连接到 I.MX6ULL | 线色建议 | 说明 |
|-----------|----------------|---------|------|
| **VCC** | 3.3V | 🔴 红色 | 电源（3.3V 或 5V 都可以） |
| **OUT** | GPIO5_IO02 | 🟡 黄色 | 信号输出（0=无物体，1=检测到物体） |
| **GND** | GND | ⚫ 黑色 | 地线 |

**接线示意图：**

```
IR 传感器                     I.MX6ULL 开发板
┌─────────┐                  ┌─────────────────┐
│         │                  │                 │
│    VCC ─┼────── 🔴 ──────→ 3.3V             │
│         │                  │                 │
│    OUT ─┼────── 🟡 ──────→ GPIO5_IO02       │
│         │                  │                 │
│    GND ─┼────── ⚫ ──────→ GND              │
│         │                  │                 │
└─────────┘                  └─────────────────┘
```

**使用说明：**
- 传感器前方有物体时，OUT 输出 **1**（高电平）
- 传感器前方无物体时，OUT 输出 **0**（低电平）
- 可以通过模块上的电位器调节检测距离

---

### 5️⃣ 板载蜂鸣器

**说明：** I.MX6ULL 开发板通常自带蜂鸣器，无需额外接线

**控制方式：**
```bash
# 通过 sysfs 控制
echo 1 > /sys/class/leds/beep/brightness  # 开启
echo 0 > /sys/class/leds/beep/brightness  # 关闭
```

**如果开发板没有板载蜂鸣器：**

可以使用外接有源蜂鸣器：

| 蜂鸣器引脚 | 连接到 I.MX6ULL | 说明 |
|-----------|----------------|------|
| **+** | GPIO（任意可用 GPIO） | 正极（需要驱动电路） |
| **-** | GND | 负极 |

**注意：** 蜂鸣器电流较大，建议使用三极管或 MOS 管驱动

## ✅ 接线检查清单

在通电前，请逐项检查：

### RC522 RFID 模块
- [ ] VCC 接 3.3V（不是 5V！）
- [ ] GND 接 GND
- [ ] RST 接 GPIO1_IO09
- [ ] SPI_CS 接 GPIO1_IO08
- [ ] SPI_SCK 接 ECSPI3_SCLK
- [ ] SPI_MISO 接 ECSPI3_MISO
- [ ] SPI_MOSI 接 ECSPI3_MOSI
- [ ] IRQ 不接

### SG90 舵机
- [ ] 棕色线接 GND
- [ ] 红色线接 5V（不是 3.3V！）
- [ ] 橙色线接 GPIO1_IO06

### IR 传感器
- [ ] VCC 接 3.3V
- [ ] OUT 接 GPIO5_IO02
- [ ] GND 接 GND

### 通用检查
- [ ] 所有 GND 已正确连接
- [ ] 电源引脚没有接反
- [ ] 接线牢固，没有松动
- [ ] 没有短路（用万用表检查）

## 🎯 通电测试步骤

### 第一步：只接电源和串口

```bash
# 先不接任何模块，只连接：
# 1. 电源线
# 2. 串口线（用于调试）
# 3. 网线（用于 SSH）

# 通电后，通过串口查看启动日志
# 应该看到正常的 Linux 启动信息
```

### 第二步：逐个连接模块

```bash
# 1. 连接 USB 摄像头
ls /dev/video*
# 应该看到 /dev/video0

# 2. 连接 RC522（先断电，接好后再通电）
# 加载驱动后检查：
ls /dev/rc522
dmesg | grep rc522

# 3. 连接舵机（先断电，接好后再通电）
ls /dev/servo
dmesg | grep servo

# 4. 连接 IR 传感器（先断电，接好后再通电）
ls /dev/ir_sensor
dmesg | grep ir_sensor
```

### 第三步：运行完整程序

```bash
# 所有模块都连接并测试通过后
/opt/visionpass/bin/visionpass
```

## ⚠️ 安全警告

### 电源安全
- ❌ **不要将 3.3V 模块接到 5V 电源**（会烧毁）
- ❌ **不要将 5V 模块接到 3.3V 电源**（工作不正常）
- ✅ 确认每个模块的工作电压后再连接

### 接线安全
- ❌ **不要在通电状态下插拔接线**（可能损坏开发板）
- ✅ 接线前先断电，接好后再通电
- ✅ 使用杜邦线时确保接触良好

### 静电防护
- ⚡ 接触开发板前先触摸金属物体放电
- ⚡ 避免在干燥环境下操作
- ⚡ 可以佩戴防静电手环
